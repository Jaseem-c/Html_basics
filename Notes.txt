HTML-hypertext Mark up language
-------------------------------
Basically Used for content creation
markup language:design purpose(programming language:logical purpose),also called tag based language,run-browser
tag-<tagname/>

types of tag
-------------
1) opening tags - <tagname>
2) closing tags - </tagname>
3)self closing tags - <tagname/> eg: br,hr,img....etc

basic structure
---------------
<html>
<head> 
- important informations , external file links,meta informations....etc
<title>title</title>
</head>
<body>
-content
</body>
</html>

6)video tag - video,source 
7)hr tag- used to create horizontal lines
8)audio tag - audio ,source

10)i frame- external web contents(nested websites)
11) break tag - br
